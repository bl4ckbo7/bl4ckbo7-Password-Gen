----------------------------ABOUT ME---------------------------


Hello the Citizens Of the WORLD! 

Bl4ckbo7 Team, We are glad to code this password generator app. 

bl4ckbo7 Password Generator is a native C++ application that is built
 
up with functionalities that help the user to generate randomly
 
passwords of their choice. This passwords MUST range from atleast 8
 
characters (which has to be the minimum password length), thus from 

there the user has the choice to generate different combinations of 

characters and numbers and symbols in his/her password.


- After generating the password, the app automatically saves the 

password in the file named "Password.txt" where the user can then copy 

and paste it in the place he want to use. Then after that he/she MUST 

delete the password file generated (For preserving the identity of 

your password).


>>System Requirements:

1. Windows 7/8/8.1/10

2. Atleast 1GB RAM

3. 1GHz CPU and later
